# Intelligent-Human-Detection-System-Under-Heavy-Smoke
本项目聚焦于“极端环境下的高鲁棒性人体检测”问题，以双模态融合为核心创新路径，研发一套基于红外图像与热成像数据融合的双模态火灾人体检测系统。该系统致力于在烟雾遮挡、高温扰动等复杂火灾现场条件下，提升人体识别的准确性和响应速度，从而为智慧消防、城市安全和应急救援提供智能化、实用化、前瞻性的解决方案。
项目以双模态数据深度融合为突破点，设计了具备多模态感知、轻量化建模和边缘部署能力的目标检测算法框架，不仅能有效提升在实际火灾场景中的鲁棒性与准确性，还能在资源受限的设备上（如消防机器人、巡检无人机等）实现高效实时推理。通过对比研究不同特征融合策略与网络结构，我们提出了一种具有创新性的轻量级双分支融合网络，并部署于瑞芯微RK3588s等主流边缘计算平台上，完成了从算法设计到实际场景应用的全链路闭环。
此外，本项目还构建了一个涵盖可见光、红外及热成像图像的高质量火灾人体检测数据集，填补了当前多模态灾害场景标注数据稀缺的空白。在模型训练方面，我们系统性地对比并调研了YOLOv8、RT-DETR、InternImage等主流检测框架的性能表现，并结合项目需求对模型进行了量化、剪枝、蒸馏等轻量化处理，兼顾性能和实用性。
从社会意义角度看，本项目可显著提升城市火灾应对的自动化与智能化水平，助力智慧城市和公共安全体系建设；从工程实现角度看，本系统具备良好的可拓展性与应用落地潜力，未来可广泛应用于工业园区、老旧社区、轨道交通等重点防火区域。通过人机协同的方式，为消防工作提供全天候、全场景、无死角的辅助感知能力，真正实现“技术赋能安全”。
